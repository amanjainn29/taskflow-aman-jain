# taskflow-aman-jain
Developed TaskFlow, a full-stack project management application where users can register, create projects, and manage tasks with features like status tracking (todo/in-progress/done), priority levels, due dates, and assignee management.

The backend was built using Go, implementing a stateless REST API with JWT authentication and PostgreSQL for relational data handling. The frontend was developed using React + TypeScript, utilizing modern tools like TanStack Query for efficient data fetching and caching. The UI was styled using Tailwind CSS and included features like optimistic updates and persistent dark mode.

The application was fully containerized using Docker, ensuring smooth setup and deployment across environments.
